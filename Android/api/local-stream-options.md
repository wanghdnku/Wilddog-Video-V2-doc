title: LocalStreamOptions
-------------------------

获取多媒体采集参数的对象。

## 方法

### isVideoEnabled()

**定义**   

```java
public boolean isVideoEnabled()
```

**说明**

本地视频流的视频开关,默认为开。



</br>

---

### isAudioEnabled()

**定义**   

```java
public boolean isAudioEnabled()
```

**说明**

本地视频流的音频开关,默认为开。

</br>

---

### getDimension()

**定义**   

```java
public Dimension getDimension()
```

**说明**

视频质量选项,默认为480P。

**返回值**

当前视频分辨率。

</br>
